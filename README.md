## Quick start
1. Install OpenClaw via the project instructions.
2. Start the gateway: openclaw gateway status
3. Use the CLI to interact: gh auth login (for GitHub actions), openclaw session_status

## Using OpenClaw
- To run a check: openclaw gateway status
- To spawn a sub-agent: openclaw sessions_spawn --task "Say hello"
